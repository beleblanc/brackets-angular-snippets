Directive options list

  • scope
  • template
  • templateUrl
  • transclude

NOTE: Options are sperated by commas (,) without spaces.

Example:

directive[scope,template,transclude] will create snippet:

directive('directive-name', [function () {
	return {
		restrict: '',
		scope: {},
		template: '',
		transclude: true,
		link: function (scope, element, attrs) {

		}
	};
}]).

	return {
		restrict: '',

		template: '',

		}
